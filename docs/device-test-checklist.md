# 真机功能验证清单

> 以下功能受模拟机能力限制，**必须在真机上验证**。模拟机无法覆盖的项已标注原因。
> 验证时建议先用登录页「生成测试数据」（`ENABLE_DEV_TOOLS=true` 时可见）准备账号与历史数据。

## 一、后台常驻监测（模拟机无法验证）

模拟机日志会报 `BussinessError 9800005: TASK_KEEPING background mode only supported in particular device`，即 `taskKeeping` 后台模式只在特定真机支持。

- [ ] 进入睡眠时段（默认 23:30–06:00，可在设置中改）后，App 切后台/锁屏，监测仍在运行
- [ ] 后台监测期间，下拉通知栏能看到常驻监测通知
- [ ] 次日 06:00（窗口结束）自动结算，弹出达标/熬夜结果通知
- [ ] 杀进程后由 WantAgent 能否被拉起（视设备策略，记录实际表现）

> 模拟机替代验证：用「今晚监测」全屏页 + 页面失焦计数那套（已验证可用），不依赖后台模式。

## 二、真实屏幕事件检测

模拟机靠 `windowStageEvent`（失焦/隐藏）近似「亮屏=离开监测页」。真机需验证真实的锁屏/亮屏广播。

- [ ] 真实锁屏 / 亮屏是否被正确计入（对比 `ScreenMonitorService` 的 commonEvent 监听）
- [ ] 下拉通知栏、左上/右上控制中心：亮屏计数 +1 且监测页/主页实时刷新（模拟机已验证刷新逻辑）
- [ ] 真实息屏放置一夜，亮屏次数是否符合预期

## 三、传感器（加速度）

- [ ] 摇一摇手机：移动次数 +1
- [ ] 左右旋转手机：移动次数 +1
- [ ] 静置不动：移动次数不增长（无误报）
- [ ] 阈值是否合适（`MOVEMENT_DELTA_HIGH/LOW`，必要时按真机灵敏度调整）

## 四、账户与数据

- [ ] 注册 → 自动登录 → 进入主页
- [ ] 记住账号 / 自动登录：重启 App 后行为正确
- [ ] 多用户隔离：A 用户的睡眠记录/周期，B 用户登录后看不到
- [ ] 改用户名（含占用校验）、改密码（含旧密码校验）
- [ ] 换头像：相册选图 → 复制进沙箱 → 头像持久化，重登仍在
- [ ] 退出登录：停止监测、回到登录页

## 五、核心玩法闭环

- [ ] 开始 30 天挑战：投入点数，主页环形进度/连续达标显示正确
- [ ] 达标次日返还 1 点；熬夜不返还
- [ ] 周期第 30 天自动结束并给出总结
- [ ] 提前结束本期：退还剩余点数
- [ ] 统计页：时长柱状图、工作日/周末洞察、点击某天看详情

## 六、边界与异常（/harden 已加固，真机回归）

- [ ] 超长用户名（20 字上限）：主页 TopBar、我的页不溢出、省略号正常
- [ ] 纯空格用户名被拒（trim 校验）
- [ ] 改名/改密/换头像快速双击不会并发重复提交（busy 防抖）
- [ ] 无数据空状态：主页空态引导、昨晚无记录、统计页无数据
- [ ] 深浅色模式切换：各页面对比度与玻璃质感正常

## 七、设置项

- [ ] 修改睡眠窗口起止时间后，主页文案与自动开始时机随之变化
- [ ] 亮屏阈值修改后，达标判定随之变化
- [ ] 通知开关
